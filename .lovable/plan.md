

# Desabilitar confirmação de e-mail

## Resumo
Configurar o sistema de autenticação para que novos usuários não precisem confirmar o e-mail antes de fazer login.

## Ação
Usar a ferramenta de configuração de autenticação para habilitar `autoConfirmEmails: true`, removendo a etapa de verificação de e-mail no cadastro.

Nenhum arquivo de código precisa ser alterado.
